The project focuses on practicing core JavaScript concepts such as functions, arrays, objects, and the Math object.

###  Functions
Functions are used to:
- Generate a random quote
- Update the content dynamically on the page

---

###  Arrays
An array is used to:
- Store multiple quotes
- Easily access and loop through the data

---

###  Objects
Each quote is represented as an object containing:
- Quote text
- Author name

This helps in organizing data in a clean and readable way.

---

## Math Object
The **Math object** is used to:
- Generate random numbers
- Select a random quote index from the array

Example:
- `Math.random()`
- `Math.floor()`
